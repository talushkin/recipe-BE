## 🚀 **אופן יבוא ל-Google Sheets:**

### שלב 1: צור Google Sheets חדש
1. גש ל-[Google Sheets](https://sheets.google.com)
2. לחץ על "גיליון אלקטרוני ריק חדש"

### שלב 2: יבא את הקבצים
**לכל קובץ CSV:**
1. **קובץ** → **יבוא**
2. **העלה** → בחר את הקובץ CSV
3. **הגדרות יבוא:**
   - סוג הפרדה: **פסיק**
   - קידוד טקסט: **UTF-8**
   - המר טקסט למספרים: **כן**

### שלב 3: צור גיליונים נפרדים
**הוסף גיליון חדש לכל טבלה:**
1. לחץ על **+** בתחתית המסך
2. שנה שם: `קטגוריות`, `חברות`, `פרויקטים`, וכו'
3. העתק את הנתונים מהיבוא לגיליון המתאים

## 🔧 **הגדרות מומלצות ב-Google Sheets:**

### עיצוב טבלאות:
```
שורת כותרת: רקע כחול, טקסט לבן, מודגש
נתונים: גופן Arial 10pt
תאריכים: dd/mm/yyyy
מטבע: ₪#,##0.00
מספרים: #,##0
```

### נוסחאות שימושיות:
```
=SUMIF(טבלת_פרויקטים!D:D,">50000",טבלת_פרויקטים!E:E)
=COUNTIF(טבלת_הצעות!H:H,"TRUE")
=VLOOKUP(A2,טבלת_חברות!A:C,3,FALSE)
```
